Why Semantic Tags?

Semantic tags are important because they allow both machines and humans to better interpret HTML. This in turn enhances the web for all users and developers.

But What About Divs?

While divs still have their use in today's web, it is considered best practice to instead use semantic tags, including:

<section>
<aside>
<header>
<footer>
<nav>
